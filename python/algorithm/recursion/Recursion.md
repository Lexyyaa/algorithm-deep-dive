# 재귀 
- 재귀는 어떠한 것을 정의할 떄 자기자신을 참조하는 것을 뜻한다 
- 프로그래밍에서 재귀는 함수가 자기 자신을 호출하는 것을 의미한다
- 재귀 함수는 일반적으로 두 가지 주요 구성 요소로 이루어져 있다 
  1. 기본 사례 (Base Case): 재귀 호출을 멈추는 조건을 정의한다 
  2. 재귀 사례 (Recursive Case): 함수가 자기 자신을 호출하는 부분이다
- 재귀함수를 사용하는 이유는 재귀함수를 이용해서 간결하교 효율성 있는 코드를 작성하기 떄문이다.
  - 예를 들어, 반복문을 사용하여 문제를 해결하는 대신 재귀함수를 사용하면 코드가 더 직관적이고 이해하기 쉬울 수 있다
  - 또한, 재귀함수는 복잡한 문제를 더 작은 하위 문제로 나누어 해결하는 데 유용하다
  - 재귀함수는 특히 트리 구조나 그래프 탐색과 같은 문제에서 자주 사용된다
  - 하지만 재귀함수는 잘못 사용될 경우 스택 오버플로우와 같은 문제를 일으킬 수 있으므로 주의가 필요하다
  - 따라서 재귀함수를 사용할 때는 기본 사례를 명확히 정의하고, 재귀 호출이 적절히 종료되도록 해야 한다
- 예제: 팩토리얼 계산
```python
def factorial(n):
    # 기본 사례: n이 0일 때 팩토리얼은 1이다
    if n == 0:
        return 1
    # 재귀 사례: n! = n * (n-1)!
    else:
        return n * factorial(n - 1)
print(factorial(5))  # 출력: 120
```

- 예제: 피보나치 수열
```python
def fibonacci(n):
    # 기본 사례: n이 0일 때 0, n이 1일 때 1을 반환한다
    if n == 0:
        return 0
    elif n == 1:
        return 1
    # 재귀 사례: F(n) = F(n-1) + F(n-2)
    else:
        return fibonacci(n - 1) + fibonacci(n - 2)
print(fibonacci(6))  # 출력: 8
```
